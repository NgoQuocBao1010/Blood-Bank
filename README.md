# Blood-Bank

### Front end (Clients) installation
<br/>

Recommend development envieronment
~~~~
Node version >= v14.17.4
NPM >= v6.14.14 
~~~~

1. Clone repo
2. Change directory to the “clients” folder
    
    ```bash
    cd clients
    ```
    
3. Install dependencies
    
    ```bash
    npm i
    ```
    
4. Run the dev server
    
    ```bash
    npm run dev
    ```
